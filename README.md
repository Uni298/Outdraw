# Outdraw Rules

## 基本概要
- **2〜6人で遊ぶ協力型パーティーゲーム**  
- 人間 vs AI の構図で、  
  **「人間には伝わるけどAIには伝わらない絵」を描く** のが目的  
- プレイヤーはスマホやタブレットをキャンバスとして使用

---

## ゲームの流れ

### 1. 描き手が秘密のお題を選ぶ
- 各ラウンドで1人が「描き手」になる  
- お題は **生き物・物体・行動・概念** など多様  
  - 例：「ネコ」「犬」「愛」「平和」など

### 2. 描き手は “AIを騙しつつ人間には伝わる絵” を描く
- AIに正しく認識されないように工夫する  
- しかし、他のプレイヤーには伝わる必要がある

### 3. AIと他のプレイヤーが回答する
- 描き手以外のプレイヤーとAIが「何を描いたか」を1回だけ推測  
- プレイヤー側には **お題のカテゴリ情報（例：生き物・概念）だけが与えられる**

### 4. 勝敗判定
- **AIが正解 → 人間チームの負け**  
- **AIが不正解で、人間の誰かが正解 → 人間チームの勝ち**

---

## 勝つためのポイント
- AIの学習データにない表現を狙う  
- 人間同士の共通認識・文脈を活かす  
- あえて抽象化したり、象徴的に描くのも有効
- 文字は禁止
